WhatWeb هي أداة تُستخدم في مرحلة **Recon / Information Gathering** لمعرفة “ماذا يعمل الموقع من خلف الكواليس”.

بمعنى بسيط:

> تكشف لك التقنيات المستخدمة في أي موقع.

---

# 🎯 ماذا تفعل WhatWeb؟

تعطيك معلومات مثل:

- 🌐 نوع الـ Web Server (Apache / Nginx)
- 💻 لغة الموقع (PHP / ASP.NET / Node.js)
- 🧩 CMS (WordPress / Joomla / Drupal)
- 🔐 تقنيات الحماية (WAF أحيانًا)
- 📦 إضافات أو Plugins مستخدمة
- 🍪 Cookies مميزة
- 🔢 إصدارات بعض الخدمات (إذا كانت مكشوفة)

---

# ⚙️ طريقة الاستخدام

```
whatweb example.com
```

---

# 🔍 ماذا يحدث عند التشغيل؟

الأداة:

- ترسل طلب للموقع
- تحلل الرد (HTTP Response)
- تبحث عن “بصمات” التقنية (Fingerprints)

---

# 📦 مثال على النتيجة

```
http://example.com [200 OK]Apache[2.4.41]PHP[7.4.3]WordPressjQuery[3.5.1]
```

---

# 🧠 شرح النتيجة

|العنصر|المعنى|
|---|---|
|Apache|نوع السيرفر|
|PHP|لغة البرمجة|
|WordPress|نظام إدارة محتوى|
|jQuery|مكتبة JavaScript|

---

# 🎯 لماذا مهمة؟

لأنها تساعدك على:

- معرفة بيئة الهدف قبل الهجوم
- تحديد الثغرات المحتملة (مثلاً WordPress قديم)
- اختيار أدوات الاستغلال المناسبة

---

# ⚡ مثال عملي أقوى

```
whatweb -v example.com
```

### `-v`

يعني:

> عرض تفاصيل أكثر (Verbose)
