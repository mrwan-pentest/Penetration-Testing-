
---
#  أداة fping 

هي أداة:

```
سريعة لفحص عدة Hosts بالـ ping
```

أفضل من ping العادي عندما:

- تريد فحص شبكة كاملة
- بسرعة

---

## فحص شبكة كاملة 

```
fping -a -g 192.168.1.0/24
```

```
fping -a -g 10.10.10.0/24 2>/dev/null
```

---

# شرح الأوبشنات

|Option|الوظيفة|
|---|---|
|-a|عرض الأجهزة الحية فقط|
|-g|توليد Range من الشبكة|
