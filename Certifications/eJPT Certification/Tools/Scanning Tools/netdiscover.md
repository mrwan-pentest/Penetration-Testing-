

أداة `netdiscover` تُستخدم لاكتشاف الأجهزة الموجودة داخل الشبكة المحلية (LAN).

تعتمد بشكل أساسي على:

```
ARP Requests
```

لذلك تعتبر ممتازة في:

- الشبكات الداخلية
- مرحلة الـ Enumeration
- اكتشاف الأجهزة الحية (Live Hosts)

---

# ماذا تفعل؟

تبحث عن:

- IP Address
- MAC Address
- أحيانًا نوع الجهاز أو الشركة المصنعة
