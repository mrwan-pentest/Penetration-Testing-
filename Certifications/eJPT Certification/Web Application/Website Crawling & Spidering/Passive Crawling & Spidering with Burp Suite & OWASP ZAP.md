
# أولاً: ما هو Crawling؟

كلمة:

```
Crawling
```

تعني:

```
عملية استكشاف الموقع وجمع روابطه وصفحاته.
```

يعني **العملية نفسها**.

مثال:

أنت دخلت موقع:

```
http://target.com
```

وتريد معرفة كل الصفحات الموجودة فيه.

عملية البحث عن الصفحات هذه اسمها:

```
Crawling
```

---

# ثانياً: ما هو Spider؟

كلمة:

```
Spider
```

تعني:

```
الأداة أو البرنامج الذي يقوم بعملية Crawling.
```

يعني:

```
Crawling = العملية
Spider = المنفذ
```
